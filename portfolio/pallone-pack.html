Pallone Pack

  • Web

Inspiration

In my school, traditionally every year, students in freshman biology make something special for their biology teacher. My friends and I decided to make a minecraft texture pack, called pallone pack, featuring the inside jokes that we all shared during the class. To advertise the pallone pack, I decided to create a website. This was all done with my teacherâ€™s permission of course.

What it does

It has a tab for pack features, showing pictures of the pack and another tab for installation instructions and links.

How I built it

I used html and css to build the website and hosted the website on github pages.

What I learned

This was my first project in which I built a website, so I learnt a lot of basic html and css. I also learnt how to host my website on github pages so that it can be public.
