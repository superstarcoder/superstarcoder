Pixel Shooter

  • Game Design
  • December 2016



PixelShooter is a fun space themed mini-game made using python's pygame module. I was inspired to develop this game since I have played a lot of games from when I was little and have always wanted to make a video game by myself. This project was not only very fun to develop, but it also helped me gain more experience in game design
